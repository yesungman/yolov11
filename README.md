# 구구단 출력 코드
```bash
for i in range(2, 10):  # 2단부터 9단까지 반복
    print(f"📌 {i}단")
    for j in range(1, 10):  # 1부터 9까지 곱하기
        print(f"{i} x {j} = {i * j}")
    print()  # 단마다 줄 바꿈
```
